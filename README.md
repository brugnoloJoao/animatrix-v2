# 🎌 AnimaTrix

O **AnimaTrix** é um catálogo dinâmico de animes desenvolvido com HTML, CSS e JavaScript puro.  
O projeto foi criado para praticar conceitos fundamentais de desenvolvimento web front-end, manipulação de DOM e lógica de programação.

---

## 🚀 Funcionalidades

✅ Catálogo dinâmico gerado via JavaScript  
✅ Cards estilizados para cada anime  
✅ Sistema de classificação indicativa  
✅ Blur automático para conteúdos restritos  
✅ Destaque especial para animes premium  
✅ Interface responsiva  
✅ Navegação simples e moderna  
✅ Efeitos visuais com hover e shadows  

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
